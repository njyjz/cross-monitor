# cross-monitor
## 功能说明
全局监控所有注册到zookeeper上的服务端以及对应的客户端，提供检测各服务端节点状态、删除失效服务端节点、查询服务端对应客户端、查询客户端对服务端请求数据、响应结果、耗时等信息的功能。
